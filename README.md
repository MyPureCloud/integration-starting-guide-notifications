# Introduction

This repository holds the projects used for Developer Starting Guides around integration using the notifications.  This projects contains three discrete projects:

1.  Integrating with notification service using a Single Page Application (SPA)
2.  Integrating with notification service using a REST-based service model.
3.  Integrating with notification service using AWS event bridge.


Each project is a standalone project and will have their own discrete instructions for setting up and running the projects.
